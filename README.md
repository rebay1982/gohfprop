# hfprop
HF Propagation Condition display for use on the command line.

## Build
`go build hfprop.go`

## Install
Just copy the binary to a directory in your path.

## Running
```
hfprof

        HF Propagation

 Band            Day     Night
-------------------------------
 80m-40m         Poor    Good
 30m-20m         Good    Good
 17m-15m         Good    Good
 12m-10m         Good    Poor
```

Super simple, no need for a web browser. Enjoy!
